# manuscripts/

Two kinds of markdown files live here:

- **Auto-generated tearsheets** under [`tearsheets/`](tearsheets/) — produced by `jellycell export tearsheet notebooks/<name>.py`. Regenerating overwrites them, so never hand-edit.
- **Hand-authored writeups** at the root of this folder — drafts, memos, papers. None in this example yet; `demo` focuses on the tearsheet flow.
