# jellycell examples

Runnable demonstration projects. Enter any directory and run:

```bash
jellycell run notebooks/<name>.py
jellycell view                         # requires [server] extra
```

| Example                              | Shows                                                                                          |
| ------------------------------------ | ---------------------------------------------------------------------------------------------- |
| [minimal](minimal/)                  | Smallest possible project — one cell, one print.                                               |
| [demo](demo/)                        | `jc.*` API tour: `jc.load` round-trip, `kind=setup` cells, implicit dep via load, JSON tearsheet. |
| [paper](paper/)                      | Research paper workflow: two figures, per-country bar chart, year-over-year, hand-authored [`paper.md`](paper/manuscripts/paper.md) alongside the auto-generated [`tearsheet.md`](paper/manuscripts/tearsheet.md). PEP-723 `[tool.jellycell] timeout_seconds` override. |
| [ml-experiment](ml-experiment/)      | Tiny regression training loop with loss curve + metrics JSON, tagged `kind=setup` hyperparams. |
| [timeseries](timeseries/)            | Multi-notebook artifact dataflow: three notebooks chained via `jc.load` on Parquet, decomposition + forecast + diagnostics, one tearsheet per notebook. |

## Tearsheets

Each non-minimal example ships a committed
`manuscripts/tearsheets/<name>.md` — a curated, markdown-native view of
the notebook that GitHub renders inline (no HTML preview service). Browse
a few:

- [`demo/manuscripts/tearsheets/tour.md`](demo/manuscripts/tearsheets/tour.md) — setup cell + JSON summary tables.
- [`paper/manuscripts/tearsheets/analysis.md`](paper/manuscripts/tearsheets/analysis.md) — two bar charts + summary table. Alongside hand-authored [`paper/manuscripts/paper.md`](paper/manuscripts/paper.md).
- [`ml-experiment/manuscripts/tearsheets/train.md`](ml-experiment/manuscripts/tearsheets/train.md) — training curve + final metrics.
- [`timeseries/manuscripts/tearsheets/03-forecast.md`](timeseries/manuscripts/tearsheets/03-forecast.md) — two plots + nested JSON flattened to a two-column table.

## The `manuscripts/` split

By convention, `manuscripts/` holds two kinds of markdown:

- **Root `manuscripts/*.md`** — hand-authored writeups (paper drafts,
  thesis chapters, decision memos). Stable across regeneration — nothing
  overwrites them.
- **`manuscripts/tearsheets/*.md`** — auto-generated by
  `jellycell export tearsheet <nb>`. Default output path. Regenerating
  overwrites.

Generate a tearsheet for any notebook with:

```bash
jellycell export tearsheet notebooks/<name>.py
# → manuscripts/tearsheets/<name>.md by default
# Override location with -o manuscripts/custom.md if you want.
```
